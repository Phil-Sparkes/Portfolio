Alt West

Alt West is a game developed in Unreal Engine by a multidisciplinary team of ten people. The game is similar to Call of Duty Zombies in where you have to fight off hordes of enemies while completing objectives. I have been placed in charge of developing the AI for the game using behaviour trees During development I have spent a considerable amount of time providing support to the other programmers and designers.

The Gobfather

The Gobfather is a game developed in Unreal. it is a town down dungeon crawler game with randomly generated levels. For this project I did the majority of the programming with exception of the UI and HUD elements.

During the prototyping phase of this project I created a map generator and refined it so that we can develop many levels very quickly. As shown in the video below.

During development of this project I experimented with putting this game into VR to see how a top down twin stick shooter would feel in this environment. I was pleasantly surprised with the result and found the game to be even more enjoyable for me at least. As shown in the video below.
